A org mode converter for Jekyll.

## Installation

Via Gemfile:

```ruby
group :jekyll_plugins do
  gem 'org-mode-jekyll'
end
```

Via gem:

``` shell
gem install org-mode-ruby
```
and add it to your _config.yml

``` yaml
gems:
  - org-mode-ruby
```
